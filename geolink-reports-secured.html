GeoLink Reports

Introduction

GeoLink Reports is the reporting platform for GeoLink (marketing DSP). We got complaints from the users (clients) about how hard it was to use GeoLink Reports. The PM was also on the same page and prioritize the problem for the quarter, so I was able to look into it.

The effort started with research, and ended up with a brand new redesign that improved the usability from above-average to excellent (SUS score from 72.5 to 91.25).

"What is GeoLink Reports?"

Existing Platform

Introduction

GeoLink Reports is the reporting platform for GeoLink (marketing DSP for agencies and brands). It offers reporting for the campaigns, the users can access online dashboard and/or download reports to understand important metrics, including pacing, performance, CTR and so on.

Online Dashboard

The users go from Start > "Select Report Category" > "Select Report" > "Configure Report Filters" > "Load Dashboard" > End

Report Download

The users go from Start > "Select Report Category" > "Select Report" > "Configure Report Filters" > "Load Dashboard" > End
